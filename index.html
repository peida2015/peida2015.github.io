SciTwits

SciTwits.com is a web application designed to be a platform for hobbyist scientists to follow and interact with other scientists. The site features custom authentication. After sign up, users can follow interesting projects and submit their own to share. They can also discuss using a live twit system and tagging system. To explore projects, users can find projects filtered by the most popular tags at the backend by clicking the tags or see all submitted projects.

Pong

Pong is a remake of the classic arcade game with Javascript. It is implemented with "initialize, update, draw" game loop pattern. The most challenging part of code is the collision detection between a circle and a rectangle.
